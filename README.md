# Weather Diary (Дневник погоды)

**Автор:** [Ваше Имя и Фамилия]

## Описание программы
Приложение с графическим интерфейсом для ведения дневника погоды. Позволяет добавлять записи о температуре и осадках, фильтровать их и сохранять историю в формате JSON.

## Функционал
- Добавление записей (Дата, Температура, Описание, Осадки).
- Валидация ввода (проверка формата даты и чисел).
- Фильтрация по дате и температуре.
- Сохранение данных в файл `weather_data.json`.

## Как запустить
1. Установите Python 3.x.
2. Скачайте файл `weather_diary.py`.
3. Запустите через терминал:
   ```bash
   python weather_diary.py
